// 工作原理

不走 API，直接用你的浏览器

从一句需求到真实结果，中间没有 token、没有注册、没有打断你正在做的事。

01
你说一句话

用自然语言描述你想做的事，或想要的工具。比如「帮我给 example.com 做个 CLI」。

02
agent 生成 / 调度 CLI

agent-cli-creator 把需求落成一个完整 CLI，或直接调度仓库里现成的 CLI 组合。

03
驱动你登录的 Chrome

CLI 通过 kimi-webbridge 操作你本地已登录的 Chrome，拿回真实页面里的真实结果。

✗ 不走 API ✗ 不折腾 token ✓ 用你真实的登录态 ✓ 装好即用
